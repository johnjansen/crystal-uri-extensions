# uri_extensions

Some simple additions to the URI class to enable the joining of uri's

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  uri_extensions:
    github: johnjansen/crystal-uri-extensions
```

## Usage

